## 运行说明
> 说明如何运行和使用你的项目，建议给出具体的步骤说明
* npm i 安装依赖
* npm link 本地添加path文件
* gfc -h 列出脚手架帮助信息
* gfc create <项目名>  创建vue cms项目
* gfc addcpn <组件名>  添加组件
* gfc delcpn <组件名>  删除组件
* gfc addview <视图名>  添加视图
* gfc addCview <视图名> -pr <父视图> 添加子视图
